# maxie-apps-homepage

Centralized public website for Maxie Apps products (MaxClicker and MaxMacro) — navigation hub, product introductions, and a unified downloads section, replacing the two standalone download pages as the primary entry point.

See [`PLAN.md`](./PLAN.md) for the phased build plan and current status.

## Development

```
npm install
npm run dev
```

## Build

```
npm run build
```
